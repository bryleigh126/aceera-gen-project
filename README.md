# 🔥 Aceera Gen Bot
A simple, fast, emoji‑powered generator bot created by Bryleigh.  
Supports multiple stock types, DM delivery, owner‑only restocking, and a 10‑minute cooldown.

This bot does **NOT** generate accounts, cracked software, or anything illegal.  
It only delivers files YOU add to the stock.

## 🛠️ Installation Guide

### 1. Create a Bot on Discord
1. Go to https://discord.com/developers/applications  
2. Create a new application  
3. Go to **Bot** → **Add Bot**  
4. Copy your **Bot Token**  
5. Enable:
   - MESSAGE CONTENT INTENT  
   - SERVER MEMBERS INTENT  

### 2. Upload Files to Waifly
Upload these files:index.js
package.json

README.md
LICENSE
